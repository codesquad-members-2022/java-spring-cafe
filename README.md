# Java Spring Cafe

백엔드 스프링 카페 프로젝트

## STEP1

> 요구사항
- 각 기능에 따른 url과 메소드 convention

| method | url             | 기능     |
|--------|-----------------|--------|
| GET    | /users/signup   | 회원가입 페이지 |
| POST   | /users/signup   | 회원가입 기능 |
| GET    | /users/list     | 회원목록   |
| GET    | /users/{userId} | 회원프로필 보기 |


> 기능 목록
- 회원가입  
- 회원목록 
- 회원 프로필 정보 보

> HTML의 중복 제거
- header, navigation bar, footer 부분에 많은 중복 코드가 있다. 중복 코드를 제거한다.
